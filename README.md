# Stats for Strava Config Tool

A single page React web application built with JavaScript and Vite for editing Stats for Strava configuration files.

### Prerequisites

- [Node.js](https://nodejs.org/) (version 16 or higher)
- [VS Code](https://code.visualstudio.com/) (recommended)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dschoepel/stats-for-strava-config-tool.git
cd stats-for-strava-config-tool
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

### Building for Production

Create an optimized production build:
```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```

### Linting

Run ESLint to check code quality:
```bash
npm run lint
```
